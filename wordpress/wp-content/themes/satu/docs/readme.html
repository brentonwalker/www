Features

Satu is built from the rock-solid Hybrid Core theme framework.

  • Ability to create custom post format.
  • Post setting that allows you to add a custom stylesheet for individual posts.
  • A smart breadcrumb menu that handles tons of permalink setups.
  • Built-in pagination for your home, archive, and search pages.
  • Post thumbnails that work with WordPress' featured image functionality and provide extra features.
  • Dynamic body, post, and comment CSS classes.
  • Fully internationalized, so you can run the theme in any language.
  • Integration with the WordPress theme customizer to allow you to customize the look of your site.
  • Ability to create custom templates for any post type in multiple ways.
  • Highly-configurable widgets that give you full control over their display.
  • Widget areas are content-aware, meaning that they only appear if they contain active widgets.
  • And much, much more!

Installation

Through Dashboard

  1. Log in to your WordPress admin panel and go to Appearance → Themes
  2. Click the Install Themes tab at the top.
  3. Type Satu in the search box and click on search button.
  4. Then Click on Install Now after that activate the theme.

Installing Via FTP

  1. Using an FTP client to access your host web server. Go to /wp-content/themes/ and make sure that you do not already have a satu folder installed. If you do, then back it up and remove it from /wp-content/themes/ before uploading your copy of satu theme.
  2. Log in to your WordPress admin panel and go to Appearance → Themes
  3. Locate the theme thumbnail and click the Activate link.

Custom Header

Satu support native WordPress custom header to change the gravatar at the top of your site. To use it, please follow the steps below:

  1. Go to Appearance → Header.
  2. Browse image from your computer and upload the image.
  3. Crop the Image to fit the size.
  4. Click on Crop and Publish.
top

Custom Background

Satu has feature to change the default background using the native WordPress custom background function. To use it, please follow the steps below:

  1. Go to Appearance → Background.
  2. Click Remove background image button to remove the default background image.
  3. You can either set the image or color as background.
  4. Click the Save Changes button when you finish.

Post Formats

Satu uses post formats feature to create a special layout for each post and provides a special options panel on your post editor. The following instructions walk you through the basics of using post formats.

  1. Click on Posts → Add New
  2. Select a Format from the box to the right of the editor.
  3. You need to enter the corresponding content depending on the format you choose.

Child theme

Since Satu is a parent theme, you'll want to create a child theme if you plan on making any customizations. Don't know how to make a child theme? It's relatively simple. Just follow the below steps.

  • Create a theme folder in your /wp-content/themes directory called satu-child.
  • Then, create a style.css file within your theme folder.
  • At the top of your style.css file, add the below information.
/**
 * Theme Name: Satu Child
 * Theme URI: http://link-to-your-site.com
 * Description: Describe what your theme should be like.
 * Version: 0.1
 * Author: Your Name
 * Author URI: http://link-to-your-site.com
 * Tags: Add, Whatever, Tags, You, Want
 * Template: satu
 */

From that point, you simply need to start adding your own styles. The parent theme styles will automatically be loaded for you.

Note that child themes are now the officially-supported way of modifying themes in WordPress.
